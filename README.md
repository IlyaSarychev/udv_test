# Тестовое UDV
## Установка

Предварительно создайте и активируйте виртуальную среду

Запуск сервера Redis:
```sh
redis-server
```

Установка зависимостей:
```sh
cd udv_test
pip install -r requirements.txt
```

Запуск приложения:
```sh
python server.py
```

Запуск клиента:
```sh
python client.py
```
